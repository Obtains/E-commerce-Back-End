## Installation
* 1. npm -u root -p
* 2. source db/schema.sql
* 3. use ecommerce_db
* 4. quit sql
* 5. npm run seed
* 6. npm start

Then open Insomina to test GET, POST, PUT, and DELETE from categories/tags/products.
